Time Capsule

Buried 2037

  • Item: A tape drive containing the full source code of GNU+Linux, specifically Debian
  • Item: The full source of GCC and a binary of GCC
  • Item: A schematic of a basic x86-64 SBC
Reasoning

Linux is included because it is the most popular FOSS OS, and the best OS for coding. The GNU Compiler Collection is a collection of compilers for many languages, including, but not limited to, GCC, for C; G++, for C++; Objective-C; and many others. These compilers could be used by future civilizations to understand our programming languages, and compile Linux. The computer schematic would be neccessary to create a computer capable of running the other programs included.

Easter Egg

An easter egg is a secret placed in a program that is supposed to be hard to find. These are put in as jokes by developers.
