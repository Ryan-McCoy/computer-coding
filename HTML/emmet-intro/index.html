Emmet in VSCode

What is Emmet

Emmet is a built-in tool for VSCode that is used primarily when editing HTML. It gives easy and powerful abbreviations and shortcuts to speed up your workflow.

Basic Usage

The most basic use of Emmet is the abbreviations. For the rest of this document, using an abbreviation means typing whatever is said then hitting tab. When you make a new HTML file, you can use the Emmet abbreviation ! to get basic boilerplate. You can also use !!! to get just the doctype declaration. To link a stylesheet, you can use the abbreviation link:css, which will link a stylesheet named style.css, which can be changed afterwards. There are many abbreviations, and generally the name of a tag can be used as an abbreviation.

Advanced Abbreviations

Abbreviations can be combined. If you want to have a div with a p tag inside, you can use div>p. The greater than symbol means to put a tag within another. For an unordered list with three elements, you can use ul>li*3. An asterisk with a number afterwards means to repeat the previous element multiple times. If you want an unordered list with three elements that each have a link and a paragraph, you can use ul>li*3>a,p. Emmet also has some more complicated abbreviations. For example, a:blank will automatically add the options to make it safe to link to an external site.

Classes and ids

With Emmet, placing a hashtag (#) after an element name will give it the id of whatever comes after the hashtag. For example, div#main will create a div with id main.
