HTML - Working With Text

Text is kind of useful

What is text?

Text is, as the name implies, text. It is generally the content in a website, although there are also images and more. Text is most often in paragraph tags, but there are other options as well.

Text tags

  • The paragraph tag (p): This represents most content on a website
  • The heading tags (h1-h6): These are all of the headings, with
  • The link tag (a): This is used for links; any text placed inside the tag will be the title of the link.

General Structure

  1. h1: The main title
  2. h2s: Major subtitles
  3. h3s: Minor subtitles
  4. Et cetera
  5. ps: The actually text content of the website.
